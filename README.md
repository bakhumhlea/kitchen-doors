Check out the prototype: http://kitchendoors.netlify.com

### Brief
Kitchen doors is a serverless [POS (Point of Sale)](https://en.wikipedia.org/wiki/Point_of_sale) web application for small restaurants. Its supported features, including checking interface, inventories tracking, and shift report. With Google's real-time database, Kitchen Doors can send and receive the data in a fraction of time, provide a better experience, and enhancing the restaurantâ€™s work-flow in a fast-paced and busy environment. 
### Technologies used
The Kitchen Doors's prototype was built using an MVC architecture framework,  **[Angular](https://angular.io/)**, alongside with its eco-system libraries; **[AngularFire](https://github.com/angular/angularfire2)** and [**Firebase**](https://firebase.google.com/).

Deployed on **[Netlify](https://www.netlify.com/)**
